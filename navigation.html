Instructions:

1. Hold down left mouse button to rotate the model. Hold down right mouse button to move the model. Use middle mouse button to zoom in or zoom out.
2. Use stack / unstack options to change the layout of the floors.
3. Use floor options to show / hide the floors
4. Control panel at the bottom: Use search button to search for a room.
5. Use movement button to move the green avatar in the space. You can also use WASD Keys to move the avatar: W = Move forward; S = Move backward; A = Move left; D = Move right; F = Go down, R = Go up.
